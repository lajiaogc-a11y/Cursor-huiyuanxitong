

## 异常报告面板优化方案

### 问题分析

当前 `ErrorReportsPanel` 直接显示原始英文错误消息和代码堆栈，存在以下问题：
- 错误消息是技术性的英文文本（如 `Cannot read properties of undefined`），非技术人员无法理解
- 没有分类和严重程度标识
- 没有修复建议
- 中英文界面下错误消息始终是英文

### 实现方案

#### 1. 创建错误分类器 `src/lib/errorClassifier.ts`

建立一个模式匹配引擎，将常见错误消息映射为：
- **分类标签**（网络错误、渲染错误、数据错误、权限错误等）
- **严重等级**（critical / warning / info）
- **双语人类可读摘要**（中文 + 英文）
- **双语修复建议**（中文 + 英文）

覆盖的错误模式（约 15 种常见类型）：

| 原始错误模式 | 分类 | 中文摘要 | 修复建议 |
|-------------|------|---------|---------|
| `Cannot read properties of undefined/null` | 数据错误 | 页面数据加载异常 | 刷新页面，如持续出现请联系管理员 |
| `Failed to fetch` / `NetworkError` / `net::ERR` | 网络错误 | 网络连接失败 | 检查网络连接后重试 |
| `ChunkLoadError` / `Loading chunk` | 加载错误 | 页面资源加载失败 | 清除浏览器缓存后刷新 |
| `401` / `403` / `permission` / `RLS` | 权限错误 | 无权限访问该资源 | 确认登录状态和账户权限 |
| `timeout` / `TIMEOUT` | 超时 | 请求超时 | 稍后重试或检查网络 |
| `JSON` / `Unexpected token` | 数据格式错误 | 服务器返回了异常数据 | 稍后重试 |
| `QuotaExceededError` / `storage` | 存储错误 | 浏览器存储空间不足 | 清除浏览器缓存 |
| `ResizeObserver` | 布局警告（低严重度） | 页面布局自动调整 | 无需处理，可忽略 |
| 其他未匹配 | 未知错误 | 发生未知异常 | 记录信息后联系技术支持 |

#### 2. 改造 `ErrorReportsPanel.tsx`

**新增显示元素：**
- **严重程度徽标**：🔴 严重 / 🟡 警告 / 🔵 信息（替代统一的红色 Error 标签）
- **分类标签**：如"网络错误"、"数据错误"等彩色 Badge
- **人类可读摘要**：替代原始错误消息作为主标题显示（跟随当前语言切换）
- **修复建议卡片**：展开详情时，在堆栈信息上方显示蓝色背景的"建议操作"区域
- 原始错误消息降级到详情展开区域内，作为技术参考保留

**UI 结构变化：**

```text
┌─────────────────────────────────────────────┐
│ 🟡 警告  网络错误  2026-02-26 10:30:15      │
│ 网络连接失败，无法加载数据                      │  ← 人类可读摘要
│ Mozilla/5.0...                               │
│                                    [🗑] [▼]  │
├─────────────────────────────────────────────┤
│ 💡 建议操作                                  │  ← 新增修复建议
│ 请检查网络连接后刷新页面重试。如果使用 VPN，    │
│ 请尝试切换网络节点。                           │
├─────────────────────────────────────────────┤
│ 原始错误信息                                  │  ← 原有技术详情
│ Failed to fetch at ...                       │
├─────────────────────────────────────────────┤
│ 错误堆栈                                     │
│ ...                                          │
└─────────────────────────────────────────────┘
```

**新增筛选功能：**
- 顶部添加按严重等级筛选的 Toggle 按钮组（全部 / 严重 / 警告 / 信息）
- 统计各等级数量

#### 3. 文件变更清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `src/lib/errorClassifier.ts` | 新建 | 错误模式匹配 + 双语分类引擎 |
| `src/components/ErrorReportsPanel.tsx` | 改造 | 集成分类器，新增摘要/建议/筛选 |

**不涉及数据库变更**，纯前端改造。
